## CSS

掌握常见的一些实现页面不可复制的方法及功能：

1. 利用 CSS，在整个页面的最上层，叠加上一层透明蒙版，让用户无法选择内容

1. 缺点：利用控制台去掉这层蒙版即可

1. 利用 CSS，给无法复制的内容添加上 `user-select: none`即可让用户无法选中内容

1. 缺点：同上，利用控制台，修改 CSS 即可
2. 但是，思考一下，这个方式在移动端其实很好用



## JavaScript

从上面可以看到，CSS 方式只能防最简单的小白，那么接下来，考虑利用 JavaScript 实现，使用 JS 的思路大致都是监听对应事件，禁止复制+剪切以及禁止右键某些选项如复制粘贴剪切：

1. 最直接的思路是将页面的复制事件跟剪切事件进行重写。除此之外，我们还能通过禁用鼠标右键菜单事件、禁止用户选中内容的方式去实现。相关代码如下所示：

```javascript
// 禁止右键菜单
document.oncontextmenu = function(){ return false; };
// 禁止文字选择
document.onselectstart = function(){ return false; };
// 禁止复制
document.oncopy = function(){ return false; };
// 禁止剪切
document.oncut = function(){ return false; };
```

也可以设置 HTML 属性标签：

```html
<body oncopy="return false" oncut="return false;" onselectstart="return false" oncontextmenu="return false">
<!--……-->
</body>
```

当然，也有破解的方式：

1. 禁止当前页面的 javascript，毕竟我们对复制拦截就是通过 js 实现的
2. 删除或者覆盖 oncopy 和 onselect 属性事件
3. OCR 技术，将图片转文字
